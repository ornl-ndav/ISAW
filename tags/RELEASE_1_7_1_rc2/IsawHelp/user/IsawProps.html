IsawProps.dat

The file IsawProps.dat is used to save a lot of user preferences. This is commonly stored in the user's home directory. The Edit Properties submenu option under ISAW's Edit menu can be used to edit this file. Any simple text editor can also be used to change this file.

    The properties in this file fall into several categories.
  1. Dimensions of IsawGUI Panel
  2. Paths for commonly used files
  3. Name and Path to external files\Data
  4. Operator and Script Search Paths
  5. Default instrument.
  6. Viewer settings
  7. Wizard settings

Operator and Script Search Paths

Key InformationValues
ISAW_HOMEThe top directory for scripts, operators ,etc. C:\ISAW
GROUP_HOMEDirectory with user scripts and operators Anywhere except Java files must be reachable via classpath
GROUPx_HOMEAnother directory for scripts and operators Same as GROUP_HOME

    The Isaw Systems searches for all legitimate scripts and operators in the following directories.

  • Script or Operators sudirectory of the directory specified by ISAW_HOME, GROUP_HOME, GROUP1_HOME, etc.

  • Any file in a subdirectory (of a subdirectory of ...) of a directory above will be searched.
