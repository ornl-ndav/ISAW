Overview

This class was developed in conjunction with the Isaw Project at Argonne
National Laboratories at Argonne ,Ill. The Project has been in process since 1998.

The AIM of the project is to interactively visualize and analyze data sets produced from the scattering of neutrons and other particle when they hit an object.

The CommandPane class has the following features:

  1. Editor Pane to Enter a sequence of Commands in either the ISAW Scripting language or Python(Jython). These commands can then be executed by the appropriate interpreter.
  2. Immediate Pane to Enter one command that can then be executed
  3. Use data sets from an invoking program
  4. Return new data sets to an invoking program
  5. A non visual Entry point for Batch File Processing
  6. An Entry point to execute Macros
The Macros can be written in the Isaw Scripting language or ,in a special format, in Java or Python( Jython).
