    Features of the language
  • It is NOT case sensitive except (presently) for the Data set Commands

    •  

     

  • There is one instruction per line

    •  

     

  • Expressions are possible anywhere a constant or variable occurs, except as the target of an assignment statement

    •  

     

  • See Commands for supported commands

    •  

     

  • true and false are supported ONLY as arguments of functions. No algebra is allowed for these.

    •  

     

  • Float constants must have a decimal point. Integers constants do NOT have a decimal point.
  • The REM statement as the first "command" on a line is considered documentation
  • Presently there are no if- then- else or subroutines structures

    •  

     

  • for loop is implemented as follows:
      •   
              For I in [1,3,5,6]
         .
         .
         .
        ENDFOR
      The list at the right can only be a list of numbers, strings, or variables and expressions.

      To denote RANGES use :. For example

      [1 : N] or [ 1 , 3 ,  8 : 15 , 2 ]

      The Ranges only work for integers and can be decreasing.

  • An ON ERROR - ELSE ERROR - END ERROR structure has been set up

    •  

     

  • A form of arrays have been implemented as follow:
      •  Res[1] is Res1

      Res[i] is Res3 if i evaluates to 3 Res[2*i+1] is Res7 if i is 3
