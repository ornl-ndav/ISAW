Arithmetic Operations :+,-, *, /, &, and ^(power)

    These binary operations(Except &) can be between Data Sets, a Data Set and a float, between two integers, between two floats, or between two strings(& only).

     Mixed types are allowed and are converted. "&" is always string add so non string data is converted to strings.

    Logical operations AND, OR, operate on two booleans and NOT is a unary operator. They are not case sensitive.
    Relational operators are >, >=, < , <= , <> , = or ==

Assignment operator =
    To store a quantity in the local workspace memory, use "=" just like it is used in FORTRAN
Load

      There are several forms of load.

    1. Load String Filename

      2. Load(String Filename)

      The names of the variables are names related to the file

    2. Load String Filename, String VariableName

      3. Load( String Filename , String VariableName )
      The names of the variables are varname[0], varname[1], varname[2], etc.

    3. Integer Load String Filename

      4. Integer Load(String Filename)
      Integer Load String Filename, String VariableName
      Integer Load( String Filename , String VariableName )
      The return value is the number of datasets loaded.
      These files are loaded into the local workspace

      NOTE: if the extension of the filename must correspond to a filetype of a supported file retriever. Currently, extensions of run, isd, xmi(ISAW xml), zip(zipped xml),sdds,gsa,gds,dat,bxs,hdf,and csd(IDEAS MC simulation).
Display
    1. Display Object Expr

      2.  The expression can be a variable or constant of any type. The data sets will be displayed in an external viewer of type IMAGE.

    2. *Display Object Expr, String ImageType

      3.  ImageType should be either "IMAGE" , "SCROLLED_GRAPHS" , "SELECTED_GRAPHS", "TABLE", or "THREE_D" without quotes(") or
        one of the following IN QUOTES, case sensitive, and with no leading and trailing spaces:
      • Contour View
      • Group x vs y
      • x vs Group y
      • x,Row vs Col y
      • Contour:Qx,Qy vs Qz
      • Contour:Qx,Qz vs Qy
      • Contour:Qy,Qz vs Qx
      • Contour:Qxyz slices
      Note: As new Viewers appear in IsawGUI.Isaw under the View menu, they can be accessed by replacing the ImageType by the name that appears on the menu item in quotes and case sensitive. Also, some of the above choices for ImageType may disappear when they are renames or eliminated.
Send
  • Send DataSet
      This command sends the data set back to anyone that is an IObserver Listener
  • Send Object Reason , Object ObservedObject
      Sends this information to all IObservers. Note order is reversed relative to the IObserver update
Save
    Save DataSet, String Filename
     This will save the data set in a form specified by the extension.

    The current extensions for saving are:

      gsa, gda, dat,isd,xmi,zip,nxs,hdj
    See Load
Return
  • Return Expression
    The program exits and the value of the Expression is returned (from the getResult Method). In scripts, this value can be assigned to another variable or used in expressions
  • Return
    The program exits and a null value is returned
