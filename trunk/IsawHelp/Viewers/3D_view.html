3D View

 

 

Introduction:

 

The ISAW 3D Viewer allows users to view detector positions and 2D slices of time-of-flight data in three-dimensional space.

 

 

 

Getting Started:

 

Launching 3D View

 

l    Before launching the 3D View, there must be data loaded in the ISAW tree.

 

l    Click on the DataSet of interest in the ISAW tree and then follow the menus:

View > 3D View

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Viewer Window:

 

 

1   3D Display

 

l    This displays the 3D view for the selected DataSet.

 

l    Each square represents a different detector position.

 

 

2   X-Scale

 

 

Time(µs) – This displays the current time range, in microseconds, used for the 3D view.  To adjust this range, double-click on a number in the readout and enter a new value.

 

Num Steps – This is used to change the number of equally spaced bins for the selected time range.

 

 

3   View Controls

 

 

Brightness – This controls the brightness of the image in the 3D display.

 

Altitude – This controls the angle above or below the horizontal plane in the viewing window.

 

Azimuth – This controls rotation around the vertical axis in the viewing window.

 

Distance – This controls the distance from the origin displayed in the viewing window.

 

 

4   Time-Of-Flight

 

 

l    This is used to step through different points in time.

 

l    Time-of-flight data displayed in each square will change as the time value changes.

 

Controls

 

           << Auto step backwards through time.

           <   Step one frame backward in time.

            ||   Pause auto step at current frame.

            >   Step one frame forward in time.

            >> Auto step forward through time.

 

 

5   Pixel Data

 

l    These readouts provide information about the currently selected pixel in the 3D display including time, the number of counts, the currently selected column/row, channel, d-spacing, Q value, Q space coordinates, energy, wavelength, and integrate.

 

 

 

Viewer Menus

 

 

File

 

Print – This option prints the visible portion of the viewer window as it appears on the screen.

 

Save As New DataSet – Select this option to save the current view as a new DataSet.  The new DataSet will appear in the tree under “Modified.”

 

Save Image – Select this option to save the visible portion of the viewer window as a JPEG picture file.

 

Save Object State – This option is not supported in the 3D view.

 

Load Object State – This option is not supported in the 3D view.

 

Close Viewer – This option closes the 3D view window.

 

 

 

Edit

 

Sum – This option is used to sum all counts of either the selected or unselected spectra.

 

Delete – This option is used to delete all counts of either the selected or unselected spectra.

 

Clear – Use this option to clear all select flags.

 

Sort by – This option is used to sort the graph data

 

 

View

 

Additional View – This option creates a duplicate 3D view window.  This new window can then be changed to a different view type.

 

Image View – This option changes the current window to an image view window.

 

3D View – This option changes the current window to a 3D view window.

 

HKL Slice View – This option changes the current window to a HKL Slice view window.

 

Contour View – This option changes the current window to a Contour view window.

 

Scrolled Graph View - This option changes the current window to a Scrolled Graph view window.

 

Selected Graph View - This option changes the current window to a Selected Graph view window.

 

Selected Table View – This option changes the current window to a Selected Table View window.

 

Table Generator – This option changes the current window to a Table Generator window.

 

Axis Conversions – This option changes the X-axis to a different unit of measurement.

 

 

Options

 

Color Scale – Use this option to select the color scale for the pixels.

 

Draw Groups – Use this option to draw an asterisk at the center of the pixel element.  The size of the asterisk is also adjustable.

 

Draw Detector Segments – This menu is used to select how detector segments are drawn in the 3D view.

 

Link Views – Select this option to link the current view with any additional views of the same data set that also have this option selected.
