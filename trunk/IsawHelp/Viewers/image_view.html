Image View

 

 

Introduction:

 

The ISAW image viewer is designed to allow users to work with a visual representation of time-of-flight data.

 

 

 

Getting Started:

 

Launching Image View

 

l    Before launching the image view, there must be data loaded in the ISAW tree.

 

l    Click on the DataSet of interest in the ISAW tree and then follow the menus:

View > Image View

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Viewer Window:

 

 

1   Image Display

 

l    This displays the image data for the selected DataSet.

 

l    Each horizontal row in the image display corresponds to a different spectrum from one detector segment.

 

l    Clicking on an area of this image will display the graph for that row in the Line Graph Display.

 

Tip: Use your keyboard's arrow keys to step through rows or columns of the image.

 

l    To Set a Select Flag:  Press s on the keyboard while a row is highlighted to set a select flag at that particular spectrum.  Repeat this step while holding Control to set additional select flags.

 

l    To Select a Range:  Press s while holding Shift.  Spectra will be selected from the first pointed at spectrum to the last pointed at spectrum.

 

l    To Zoom:  Click and drag a box around the area of interest using the middle mouse button.  If no middle mouse button is available, hold Alt while dragging with the left mouse button.  Double click any mouse button to reset the zoom.

 

 

 

2   Line Graph Display

 

l    This displays the intensities for the currently pointed at row in the image view.  Lines in this graph are drawn with a vertical scale determined by the auto-scale control setting.

 

 

 

3   X-Scale

 

 

Time(µs) – This displays the current time range, in microseconds, used for the image view.  To adjust this range, double-click on a number in the readout and enter a new value.

 

Num Steps – This is used to change the number of equally spaced bins for the selected time range.

 

 

 

4   View Controls

 

 

Brightness – This controls the brightness of the image in the image display.

 

Auto-Scale – This controls the scale of peaks seen in the line graph display.  An auto-scale feature, which fits the highest peak into the display, is used when the slider is positioned at the far left side of the scale.  The scale can be manually adjusted by moving the slider to the right.

 

 

 

5   Cursor Readout Area

 

 

Image Data – These readouts provide information about the currently selected point in the image display including time, the number of counts, the currently selected column/row, channel, d-spacing, Q value, Q space coordinates, energy, wavelength, and integrate.  The readouts only respond when the cursor is in the image area.

 

Graph Data – These entries provide similar information about the line graph display.  The readouts respond when the cursor is in the graph area or image area.  If Graph Cursor Tracks Image Cursor is unselected under the Options menu then these readouts will only respond when the cursor is in the graph area.

 

 

 

Viewer Menus

 

 

File

 

Print – This option prints the visible portion of the viewer window as it appears on the screen.

 

Save As New DataSet – Select this option to save the current view as a new DataSet.  The new DataSet will appear in the tree under “Modified.”

 

Save Image – Select this option to save the visible portion of the viewer window as a JPEG picture file.

 

Save Object State – This option is not supported by the image view.

 

Load Object State – This option is not supported by the image view.

 

Close Viewer – This option closes the image view window.

 

 

Edit

 

Sum – This option is used to sum all counts of either the selected or unselected spectra.

 

Delete – This option is used to delete all counts of either the selected or unselected spectra.

 

Clear – Use this option to clear all select flags.

 

Sort by – This option is used to sort the graph data.

 

 

View

 

Additional View – This option creates a duplicate image view window.  This new window can then be changed to a different view type.

 

Image View – This option changes the current window to an image view window.

 

3D View – This option changes the current window to a 3D view window.

 

HKL Slice View – This option changes the current window to a HKL Slice view window.

 

Contour View – This option changes the current window to a Contour view window.

 

Scrolled Graph View - This option changes the current window to a Scrolled Graph view window.

 

Selected Graph View - This option changes the current window to a Selected Graph view window.

 

Selected Table View – This option changes the current window to a Selected Table View window.

 

Table Generator – This option changes the current window to a Table Generator window.

 

Axis Conversions – This option changes the X-axis to a different unit of measurement.

 

 

Options

 

Color Scale – Use this option to select the color scale for the image display.

 

Graph Selected – Use this option to graph the selected graph data; up to 16 selected line graphs can be graphed.  This option is on by default.

 

Graph Cursor Tracks Image Cursor – Select this option to allow graph data readouts to respond when the cursor is in the image display.

 

Horizontal Scroll – Select this option to make every vertical line on the graphs represent the same time bin.  This allows for viewing a long range of time values accurately.  If left unselected, bins will be compressed to fit inside the view area.

 

Graph Rebinned Data – Select this option to rebin data automatically so that each vertical line represents the same time.

 

Link Views – Select this option to link the current view with any additional views of the same data set that also have this option selected.  This is on by default so that pointing the cursor in one view affects all views of the same data.
