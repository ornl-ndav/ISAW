Overview

    This class was developed in conjunction with the Isaw Project at Argonne National Laboratories at Argonne ,Ill. The Project has been in process since 1998.

    The AIM of the project is to interactively visualize and analyze data sets produced from the scattering of neutrons and other particle when they hit an object.

    The CommandPane class has the following features:

    1. Editor Pane to Enter a sequence of Commands that can be run.

    2. Immediate Pane to Enter one command that can be run

    3. Use data sets from an invoking program

    4. Return new data sets to an invoking program

    5. A non visual Entry point for Batch File Processing

    6. An Entry point to execute Macros
