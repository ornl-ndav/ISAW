ISAW Parameter GUI Help


PARAMETER

DESCRIPTION

EXAMPLE
<ArrayPG



 ArrayPG produces a text field that allows the user to enter values into an array. The result is a vector.

$x Array(2.00,5.00,10.0) Enter Values

Display x
BooleanPG



 BooleanPG is a class that produces the checkbox GUI element for specifying true or false. The result is boolean.

$x Boolean Enter x

Display x
ChoiceListPG







 ChoiceListPG is a class that produces the drop-down menu GUI element containing a list of values to choose from. The return value can be either a string or a numerical value depending on the input list. For example, if a list of strings is input then the return value will be a string.

$x ChoiceList( ["a","b","c","d"]) Select Value

Display x
DataDirPG



 DataDirPG produces a directory chooser window that is used for selecting a directory. The result is a string.

$x DataDir Select Directory

Display x
DataSetPG




 DataSetPG produces a drop-down menu containing a list of DataSets that are currently in the tree. The result is a data set.

$x DataSet Select DataSet

Display x
FloatArrayArrayPG




 FloatArrayArrayPG is used to create a medium-sized, two-dimensional float array. The result is a vector of vectors of float values.

$x FloatArrayArray Enter Values

Display x
FloatArrayPG



 FloatArrayPG is used to create a medium-sized, one-dimensional float array. The result is a vector of float values.

$x FloatArray Enter Values

Display x
FloatPG



 FloatPG produces a text field that is used to enter data containing a floating point value. The result is a float.

$x Float(1.5) Enter Value

Display x
FuncStringPG




 FuncStringPG produces a text field that is used to enter a string that represents a mathematical function. The result is a string.

$x FuncString(“This is a string”) Enter String

Display x

InstNamePG








 		InstNamePG produces a text field that is used to enter instrument names. If the initial value is left blank then the default instrument will be used. The default instrument can be specified in ISAWprops.dat under "Viewer Options" by editing the value for Default_Instrument. The result is a string. $x InstName(“GPPD”) Enter Instrument

Display x
IntArrayPG








 		IntArrayPG produces a text field that is used to enter an increasing sequence of consecutive integers. Its value is stored as a formatted string. Individual values are separated by a comma and a set of consecutive values can be represented by inserting a colon between the start and end values. The result is a string. $x IntArray(-1,0,1,10:50) Enter Values

Display x
IntegerArrayPG




 		IntegerArrayPG produces a list box with editing buttons. It is used to create medium-sized, one-dimensional integer arrays. The result is a vector of integers. $x IntegerArray ( [1,3,5] ) Enter Values

Display x
IntegerPG



 		IntegerPG produces a text field that is used to enter integers. The result is an integer.

$x Integer(-1) Enter Integer

Display x

LoadFileArrayPG




 		LoadFileArrayPG produces file chooser windows that allow the user to choose a list of files from multiple directories. The result is a vector containing file names.

$x LoadFile(“C:/isaw/SampleRuns/GPPD1.run”)

Display x

LoadFilePG



 		LoadFilePG produces a file chooser window that allows the user to choose a single file. The value is a string.

$x LoadFile(“C:/isaw/SampleRuns/GPPD1.run”)

Display x

MaterialPG






 		MaterialPG produces a text field that is used to enter material names. Chemical formulas must be typed according to the example to the right using commas and underscores. The result is a string.

For Trifluoromethanesulfonic acid

(CHF3O3S)

$x Material(C,H,F_3,O_3,S) Enter Material

Display x

MonitorDataSetPG




 		MonitorDataSetPG produces a drop-down menu that contains a list of available monitor data sets. The result is a data set.

$x MonitorDataSet Select DataSet

Display x

PlaceHolderPG





 		This ParameterGUI saves an opaque object value which cannot be altered by the GUI. Parameters can be passed between operators but cannot be modified by the scripting language.

$x PlaceHolder("abc") Enter x

Display x

PrinterNamePG




 		PrinterNamePG produces a drop-down menu that contains a list of printer names available to the system. The result is a string.

$x PrinterName(\\server\HP Color LJ4500)

Display x

PulseHeightDataSetPG




 		PulseHeightDataSetPG produces a drop-down menu that contains a list of available pulse height data sets. The result is a data set.

$x PulseHeightDataSet Select DataSet

Display x

Qbins1PG


 		Qbins1PG produces a window with text fields that allow the user to enter start, end, and number Q bins for a sublist. Constant dQ or dQ/Q choices are also supported. The result is a vector of floats.

Note:

Format follows this example:

( [ Start Value, End Value,

 Number of Steps, Constant dQ or dQ/Q] )

True = Constant dQ

False = Constant dQ/Q


$x Qbins1( [1,9,5,true] ) Enter Values

Display x

QbinsPG







 		QbinsPG produces a list box with editing buttons that allows the user to enter a large list of Qbin boundaries with either constant Q widths or constant ratios. This Parameter GUI allows for concatenating of several of these lists. The result is a vector of floats.

$x Qbins( [1,9,5] ) Enter Values

Display x

RadioButtonPG



 		RadioButtonPG is a class that produces radio button GUI elements. The result is a string.

$x RadioButton( [“a”,”b”,”c”] ) Enter x

Display x

RealArrayPG






 		RealArrayPG produces a text field that allows for entry of small lists. The input can be multi/uni dimensional arrays of integers, floats, doubles, or strings. The result is a multi-dimensional array that depends on the initial value.

$x RealArray([ [1,2,3], [4,5,6] ] ) Enter Values

Display x

SampleDataSetPG



 		SampleDataSetPG produces a drop-down menu that contains a list of available sample data sets.

$x SampleDataSet Select DataSet

 		SaveFilePG produces a file chooser window that allows the user to choose a single file in a directory. The value is a string.

$x SaveFile Select File

StringArrayPG




 		StringArrayPG produces a list box with editing buttons that allows users to enter one-dimensional string arrays. The result is a vector of strings.

$x StringArray( [“abc”,”def”] ) Enter Values

Display x

StringPG



 		StringPG produces a text field that is used to enter a string. The result is a string.

$x String(“This is a string”) Enter String

Display x

UniformXScalePG UniformXScalePG produces a command pane that allows the user to specify a start value, an end value, and the number of X values for a uniform X scale. The result is a vector of float values.

Note:

Format follows this example:

( [ Start Value, End Value,

Number of X Values ] )


$x UniformXScale( [1,9,5] ) Enter Values

Display x

VariableXScalePG




 		VariableXScalePG produces a list box with editing buttons that allows the user to enter arbitrary X scales. The return value is a vector of floats.

$x VariableXScale Enter Values
