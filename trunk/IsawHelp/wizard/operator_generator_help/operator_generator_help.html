Introduction:


The Method to Operator Wizard is designed to allow users to easily create custom operators from existing methods and store them for use in ISAW.



Getting Started:



Launching from a command line

To launch the Method to Operator Wizard from a command line, type the following.

java devTools.Method2OperatorWizard



Tabs:


Information Tab

The information tab is used to enter data about the operator’s creator. This data will automatically be included as part of the documentation for the operator’s code. Information entered in this tab can also be stored in a separate file for use in other operators by choosing Save Contact Info under the File menu.


Your Name – Enter the name that you would like to appear in the operator.

Your Email – Enter the email that you would like to appear in the operator.

Institution – Enter the name of the institution that you would like to appear in the operator.

Address – Enter the address that you would like to appear in the operator.

Acknowledgements – Enter any acknowledgements that you would like to appear in the operator

Note: The acknowledgements section should include the following information.

This work was supported by the Intense Pulsed Neutron Source Division of Argonne National Laboratory, Argonne, IL 60439-4845, USA.

This work was supported by the National Science Foundation under grant number DMR-0218882.



Method Info Tab

Select Class with Static Method – Press the Browse button to find the Java class that contains the static method you would like to use. Press the View File button to view the Java code for the selected class in a new window.

Select Method – Use the drop-down menu to choose a method to work with.

Set GUI Info for Each Parameter – Use this area of the Method Info tab to adjust GUI info for each of the method’s parameters.


Arguments – The arguments box lists all of the available parameters for the currently selected method.

To Edit Gui info: Select a parameter under Arguments. Then enter information into the text fields following Prompt, Param Name, and Init Value.

Param GUI – Use the Param GUI button to view information about the currently selected parameter GUI.

Param GUI Menu – This menu lists all parameter GUIs that represent the selected argument.




Operator Info Tab


Select Operator Output File – Press the Browse button to select a file to save the completed operator to.

Note: This does not save the current session.

Set Operator Descriptions – Use this area to edit the operator’s properties.

Operator Title – Enter a title for the operator; this title will be used as the operator’s name in all menus.

Command for Scripts – Use this text field to describe how the operator is used in scripts.

Category – Use this text field to specify the menu location of the operator. Operators will be placed under the Macros menu. Any additional sub-menus that are specified will be created if they do not already exist.



Documentation Tab


Overview – Enter a brief description of the operator in the overview text box.

Assumption – Address any assumptions within the operator. This includes special data set requirements, positive/negative values, and units of measurement.

Algorithm – Enter information about the mathematical processes used in the operator.

Return – Use the return text field to enter information about the operator’s return values. Be sure to include information about the return type and units of measurement.

Parameter – Use the parameter text field to describe the operator’s variables and stored values.

Error – Use this text field to describe any known errors that exist within the operator. Exceptions will be translated into error messages as specified by this field.



Menu


File

Load Contact Info – Load your saved contact information into the information tab.

Load Session – Load your saved session information into all of the tabs.

Create Operator – Create the currently defined operator and save it to the file chosen in Select Operator Output File under the operator info tab.

Note: This does not save the current session.

Save Contact Info – Save the data contained in the information tab as a separate file for use in other operators.

Save Session – Save the current session information from all tabs as a separate file for later editing.

Exit – Exit the Method to Operator Wizard.
