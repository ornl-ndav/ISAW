Selecting Peaks/Points in Q Space

This button presents options to select peaks(or points) and specify hkl values.
These selected points are arguments to several calculate orientation matrix routines and to at least one view in QViewer option.
    Notes on format
  • The options are set up to easily specify one peak at a time selected from the 3D QViewer of the Peaks.

  • Since most options that use selected points require 4 or fewer selected points, the menu gives options to select Peak1, Peak2, .. Peak4.

    Sub menu options
  • Set/Alter Peak- Submenu options are to setPeak1, setPeak2,..,setPeak4, then set Other Peaks(up to 12 peaks). For each option a dialog box pops up where the peak number(if over 4), use peak selected in the QViewer, specify sequence number(if previous choice is false), and specify Q values( if a sequence number is less than 0).NOTE:Make sure dialog box has "Enter Q Vals"

  • Set/Alter hkl Value- Submenu options are to setPeak1, setPeak2,..,setPeak4, then set Other Peaks(up to 12 peaks). For each option a dialog box pops up where the peak number(if over 4), use peak selected in the QViewer,and specify hkl values if Use Selection from QViewer is not selected. NOTE:Make sure dialog box has "Enter h,k,l Vals"

  • Set List of Peaks - Allows for entering a large( can be more than 12) list of sequence numbers all at once.

  • Clear All Set Peaks-Clears all selected peaks.
